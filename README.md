# python-phase0-foundation

## Phase0 について

- Phase0 は Python の「基礎筋トレ」フェーズです。
- `phase0/` 以下のドリルは、必ず自分の手でコードを書いてください。
- 解答をすぐに見るのではなく、「なぜそうなるか」を考えながら試行錯誤することを目的としています。
- このリポジトリには模範解答コードは含めません。

### 進め方の一例

1. `phase0/p0_01_hello_basic.py` から順に読み、docstring の問題文と要件を確認する。
2. `pass` と書かれている部分を自分で実装してみる。
3. `pytest` を実行し、`tests/` 以下の仮テストが通るように調整する（最初はわざと失敗するようになっています）。
4. 動いたら、少し書き方を変えてみるなどして理解を深める。

> 大事なルール: Phase0 では「コピペで終わらせない」。必ず自力で書くこと。
